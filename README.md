# Sistem Peminjaman Ruangan

Aplikasi berbasis web untuk mengelola peminjaman ruangan kelas dan laboratorium di Politeknik Negeri Lampung.

## 📌 Fitur Utama
- Login & Register User
- CRUD Data Peminjaman (Tambah, Edit, Hapus)
- CRUD Data Ruangan
- Approve / Persetujuan Peminjaman (Admin)
- Cetak Bukti Peminjaman

## 🛠️ Teknologi yang Digunakan
- PHP (Backend)
- MySQL (Database)
- HTML, CSS (Frontend)
- XAMPP (Localhost Server)

## 🚀 Cara Menjalankan
1. Install XAMPP
2. Jalankan Apache & MySQL
3. Pindahkan folder project ke:
   htdocs/
4. Buka browser:
   http://localhost/peminjaman_ruangan
